# Blackjack

## Objektorienteeritud programmeerimine

Autorid: Elias Mikael Teikari, Oliver Iida, Karl Elmar Vikat

## Klassid

Main.java - käivitab programmi ja alustab blackjacki mängu.
BlackjackMäng.java - juhib mängu käiku ja otsustab, kes võidab.
Kaardipakk.java - hoiab kaardipakki, segab seda ja jagab kaarte.
Kaart.java - kirjeldab ühte mängukaarti.
Mängija.java - hoiab mängija andmeid, kaarte ja punktisummat.
